Global Development and Environment Institute (GDAE)

Technical Director, GDAE, U.S.A (07/2009 - 02/2010)

My primary function in GDAE was to Lead a TEAM to create an online Social Science Library (SSL) using the opensource software GREENSTONE. The library has the ability to search for articles with a multiple combination of attributes. Some of the sample images below show the different categorical searches capable by the software. I finalized the design, created a GSDL file which would compile the articles under the diffrent categories and made it available online. The software was completed in time on budget. It is currently being distriuted online and via CDs and USB thumb drives.
