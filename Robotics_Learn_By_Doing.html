Robotics short overview...

Module 1: Electricity and Electronics

The module is a groundwork to for building a robot in future modules. I have built and tested the basic analog switching circuits, light flashers, sound effects and controlling servo motors. The circuits were built and tested using resistors, capacitors, pots, LED's, relay SPDT switches, 7803, 7805 voltage regulator, IRFZ44N MOSFET power transistors, LM386 audio amplifier, TL074 op amp, hobby servo motors.

Module 2: Digital Electronics

In this module I have built digital electronic circuits. We use the PIC microcontrollers and program the same using MPLAB. The course ends in the building and programming of a Arduino smart car. The parts used to build are PIC16F1455 microcontroller. PICKIT 3 programmer, Arduino TFT touch screen shield, Hitachi LCD display, Arduino Uno with the ATMEGA328P.

Module 3: Robotic Drives and Physics

This module delves into physics principles with the goal of designing drive systems for robots. I gained some practial knowledge about timing belts, industrial robots, DC brushless servo motors.
